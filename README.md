# AutoExp

AutoExp is a turn-key setup for autonomous ML experimentation. It provides a simple and effective way to run multiple experiments and track their results.

## One-time step up in your repo

In your repo, send your coding agent such as Codex or Claude Code this message:
```
Read https://github.com/wizwand/autoexp/blob/main/make_autoexp.md and follow the instructions to create a file `program.md` in the current directory.
```

## Run automated experiments

In your repo, send your coding agent such as Codex or Claude Code this message:
```
Hi have a look at program.md and let's kick off a new experiment!
```
